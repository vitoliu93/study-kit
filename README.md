# study-kit

自学工具包 plugin。北极星 + 六项决议:`docs/north-star.md`。

**北极星:让下一次学习会话更容易发生。** 姿态:有经验的导师,不是听话的助手(L2 主动纠偏 + 单主线红线 L3)。

- **resume-learning** — 学习存档/读档。学完说"存档"(取证 + 追问式主动回忆 + AI 代写 RESUME.md);想学说"读档"(30 秒回顾 + 下一步)。重启成本恒为 30 秒。
- **study-coach** — 第一性原理学习教练。目标审计(五步法砍材料)/ 迷茫急救(诊断阻力开 15 分钟处方)/ 进度分析(存档历史的证据式分析)。
- **quiz-me** — 概念练习出题人。说"出题/练一下 <概念>"触发:情境应用题不出术语名,考"识别该用什么概念";一题一判按骨架定位断点,transfer 阶梯从熟悉走向陌生(生活场景→回到材料→边界陷阱),高阶反向出题。一次 3 题封顶。

生长规则:不设冻结期,但工具迭代必须伴随学习行为发生;新功能先过北极星裁决(增加下次启动成本的一律否决)。
